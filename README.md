## 🐍 Conda Installation

Install via `conda`:

```bash
conda install salvatorecalderaro::geco-explainer
```
## 🐍 Conda (via environment)

You can also set up a Conda environment and install via pip:

```bash
conda create -n geco python=3.10
conda activate geco
pip install geco-explainer
```
